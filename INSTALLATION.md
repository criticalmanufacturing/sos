# Installation of cmf-sos

To install cmf-sos all you have to do is run this command on your terminal

`curl -OL https://github.com/criticalmanufacturing/sos/raw/refs/heads/main/installation/install.sh`

Then just run it

`chmod +x install.sh && ./install.sh` 

And that's it! A message should appear saying:

`cmf-sos installed successfully. You can now run 'cmf-sos' to start the application.`
